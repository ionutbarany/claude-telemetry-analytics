# Claude Telemetry Analytics

Production-style telemetry analytics platform.

## Structure

- `app/` — FastAPI application (API, core, db, services)
- `etl/` — parsers, transformers, and loaders
- `dashboard/` — Streamlit pages and components
- `tests/` — pytest suite
- `docs/` — project documentation
- `docker/` — container-related assets
